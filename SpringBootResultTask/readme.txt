Стартовая точка запуска приложения - "SpringBootResultTask/src/com/notnetcracker/start/SpringBootStarter
Старатовая страница "http://localhost:8080/index"

По умолчанию все добавленные Person сериализуются в json и добавляются в файл "resources/userfiles/ListOfPerson.json".
При запуске проекта происходит автоматическая десериализация Persons из файла, а так же их добавление в общий список Persons.
Реализованы возможности добавить человека ( /addPerson ) и просмотреть всех добавленных людей (/personList)

Добавлена валидация на вводимые данные ( /addPerson )
В поле id можно вводить только целые натуральные числа
В поле First name, Second name и Work Place можно вводить только буквы латинского алфавита
В поле salary можно вводить только числа ( в т.ч числа с плавующей точкой) P.S. Разделять через точку (125.251)
В поле Mail вводится текст в формате ["name" @ "domenMailName" ]
Так же, теперь нельзя оставить какое-либо поле незаполненным.

При запросе ( /findPerson ) происходит поиск Persons по firstName и lastName, вывод найденых Persons,
а так же информация о дате обращения и информация о браузере, с которого был выполнен запрос.
